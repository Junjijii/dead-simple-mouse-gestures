Privacy Policy

Dead Simple Mouse Gestures — Chrome Extension by darumastudio

Last updated: March 2026

Data Collection

Dead Simple Mouse Gestures does not collect, transmit, or share any personal data. No analytics, no tracking, no external requests of any kind.

Local Storage

The extension uses Chrome's local storage API solely to save your gesture key bindings on your own device. This data never leaves your browser.

Permissions

The extension requests only the permissions necessary to function:

  • tabs — to close the current tab on downward gesture
  • activeTab — to reload the current tab on upward gesture
  • storage — to save settings locally
  • sessions — to navigate browser history (back/forward)

If you have any questions, please open an issue on GitHub.
